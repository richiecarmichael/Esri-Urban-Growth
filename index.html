About The App

This application was designed and developed by Esri's Applications Prototype Lab in Redlands, California. The application is built using the following technologies:

  • ArcGIS API for JavaScript (beta) by Esri
  • D3.js by Mike Bostock
  • Google Fonts, specifically Roboto and Berkshire Swash
  • jQuery by the jQuery Foundation Inc.

Earthquake data consists of 10,835 seimic events from 1900 to July 2015 with magnitudes of six or greater on the Richter scale. The data was downloaded from the USGS's ANSS Comprehensive Earthquake Catalog. Information on the largest and deadliest earthquakes was sourced from Wikipedia.

Seismic events hightlighted red in the chart indicate those events within a one year period, or half a year either side of the red line.
